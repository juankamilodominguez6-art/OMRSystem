# 🏪 Sistema de Punto de Venta (POS) Profesional

Sistema completo de POS para tiendas de abarrotes y fruver, con interfaz de escritorio moderno.

## 📋 Características Principales

### 🔐 Módulo de Inicio de Sesión
- Roles de usuario: Administrador, Empleado, Gerente, Invitado
- Control de accesos por roles
- Sesiones seguras

### 💻 Panel de Ventas (POS)
- Teclado numérico virtual en pantalla
- Navegación por categorías (Frutas, Granos, Licores, etc.)
- Catálogo de productos con imágenes, descripciones y precios
- Carrito de ventas interactivo
- Múltiples métodos de pago (Efectivo, Tarjeta, Crédito)
- Cálculo automático de devuelta/cambio
- Emisión de recibos personalizables (58mm/76mm/80mm)

### 📊 Administración y Backend
- Gestión de Clientes (limites de crédito, contactos, direcciones)
- Gestión de Proveedores
- Inventario completo (entrada/salida, stock bajo, reportes)
- Caja y Finanzas (movimientos, cierre diario)
- Reportes y Estadísticas (ventas, productos, gráficos)
